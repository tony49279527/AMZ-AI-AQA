# 本地启动说明

## 1. 在你自己电脑上打开终端

用 **终端 / Terminal**（不是 Cursor 里让我跑命令），在项目目录下执行。

## 2. 进入项目目录并启动

```bash
cd "/Users/liangxile/project/AMZ AQA /report-generation-system"
pnpm dev
```

或如果已经在 `report-generation-system` 目录下：

```bash
pnpm dev
```

## 3. 看到类似输出后

```
▲ Next.js 16.0.10 (Turbopack)
- Local:    http://127.0.0.1:3000
✓ Ready in ...
```

用浏览器打开：**http://127.0.0.1:3000** 或 **http://localhost:3000**

## 4. 若仍打不开

- 确认终端里没有报错、且一直停在 `Ready` 没有退出。
- 换浏览器试试（Chrome / Edge / Safari）。
- 确认没有别的程序占用 3000 端口；若占用可改端口：  
  `pnpm exec next dev -H 127.0.0.1 -p 3001`  
  然后访问 http://127.0.0.1:3001
